The parameters for the template configuration are the following :

  • name: string, name of the template
  • source: string, source of the template (optional)
  • text: list, list of text inputs, will be detailed below
  • example: list, example use of the template (optional, will not be detailed, see example config.yml)
  • overlay: list, if you want to overlay an image on top of your background (like to change the face of a person in the meme)

Let's detail the parameters for each text element:
  • style: string, accepts default or upper
  • color: string, font color (black or white basically)
  • font: string, accepts thick, thin or comic
  • anchor_x: float, x coordinate of top-left corner of textbox. 0.0 is leftmost pixel of the image, 1.0 is rightmost pixel of the image
  • anchor_y: float, y coordinate of top-left corner of textbox. 0.0 is top pixel of the image, 1.0 is bottom pixel of the image
  • angle: float, text rotation. 0.0 is normal, 180.0 is upside down (rotated counter-clockwise), and -180.0 is upside down but rotated via the other way (clockwise)
  • scale_x: float, x size of textbox. 0.0 is null, 1.0 is full width of the image
  • scale_y: float, y size of textbox. 0.0 is null, 1.0 is full height of the image
  • align: string, alignment of text in textbox. Most likely you want center
  • start: float, if you want text to be animated, defines when text appears on gif. 0.0 is at frame 0, 1.0 is last frame
  • stop: float, if you want text to be animated, defines when text disappears on gif. 0.0 is at frame 0, 1.0 is last frame

As an example, here is a config.yml file for the vince template: 
name: Vince McMahon Reaction
source: https://knowyourmeme.com/memes/vince-mcmahon-reaction
text:
  - style: default
    color: black
    font: thin
    anchor_x: 0.0
    anchor_y: 0.0
    angle: 0.0
    scale_x: 0.495
    scale_y: 0.315
    align: center
    start: 0.1
    stop: 1.0
  - style: default
    color: black
    font: thin
    anchor_x: 0.0
    anchor_y: 0.32
    angle: 0.0
    scale_x: 0.495
    scale_y: 0.357
    align: center
    start: 0.4
    stop: 1.0
  - style: default
    color: black
    font: thin
    anchor_x: 0.0
    anchor_y: 0.68
    angle: 0.0
    scale_x: 0.495
    scale_y: 0.32
    align: center
    start: 0.7
    stop: 1.0
example:
  - 2-Day Delivery
  - Overnight Shipping
  - Shopping at the Store
overlay:
  - center_x: 0.25
    center_y: 0.155
    angle: 0.0
    scale: 0.3
  - center_x: 0.25
    center_y: 0.5
    angle: 0.0
    scale: 0.3
  - center_x: 0.25
    center_y: 0.84
    angle: 0.0
    scale: 0.3
